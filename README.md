# 했음 리스트

## 목표 달성을 얼마나 진행했는지 알 수 있게 도움

예시) 
### 1. 아침,저녁약을 먹기로 목표를 설정하고 선지에 먹음/안먹음 추가해서 실제로 먹었는지 체크

### 2. 요리를 만들때 목표 요리를 설정하고 레시피 재료를 선지로 설정하면서 실제로 재료를 사용했는지 체크

### 3. 목표 일정을 설정하고 그 일정을 했는지 안했는지를 선지로 설정하기

## 사용법
1. 홈화면의 +목표추가 버튼 터치
2. 빈칸을 목적에 맞게 잘 채우고 저장
3. 좌측 상단에 햄버거 버튼으로 페이지 이동할 수 있음
4. 기록 탭에서 여태 목표를 몇회 달성했는지, 몇회 선지를 선택했는지 알 수 있음.
